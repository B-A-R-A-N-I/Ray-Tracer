# Ray Tracer

A simple ray tracer written in C++. This project uses **Anti-Aliasing** and **Defocus blur(Depth of Field)** features to render the scene. The scene is rendered with different camera angles with same **Field of View(FOV)** to note where the camera focuses based on different angles.
